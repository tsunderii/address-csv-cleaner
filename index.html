Address CSV Cleaner

Clean and format raw US addresses to USPS standards using Gemini.

How to Use This App

  1. Enter your Google AI Studio API key (click "Get API Key" if you don't have one).
  2. Select or drag‑and‑drop your CSV file containing raw addresses.
  3. Click Process CSV to clean the data.
  4. Download the cleaned CSV using the link that appears.
